Co Branding

Usage

This title variant is for use when a page requires a supplementary branding logo, to preserve the hierarchy between the main University of Cambridge branding and any additional graphic treatment.

This variant should be used judiciously and sparingly across the University, to reduce confusing the user by presenting them with multiple, potentially conflicting brand logos.
