Styleguide notes

Tooltip

Usage

The tooltip is used to provide additional information when rolling over, for example, an image or icon.

We recommend using this component sparingly as tooltips are not really appropriate for responsive structures supporting small screens.

Interaction

N/a

CSS and media queries

N/a

Use of Javascript

Tooltips require the addition of the Tooltip library which detects the height and width of the page to place the tooltip in the center of the screen regardless of screen width.
