Styleguide notes

Tabbed navigation and pill navigation

Usage

This component provides an easy way for users to switch between different panels of related information.

The pills style is the most flexible option to accommodate potentially either many or few tabs, with varying character lengths.

The same solution applies at any screen widths, and the pills wrap as required - ie more than one pill can appear in a given row.

Interaction

N/a

CSS and media queries

N/a

Use of Javascript

The pills and tabs require the Twitter Bootstrap Javascript library to function. Each widget requires explicit declaration inside the DOM ready event.

Component variants

We recommend using tabs sparingly as the horizontal layout does not lend itself to viewing on the mobile platforms.
