Inline images set to scale

Images will expand in size until they reach their full dimensions. Images larger than their containing div will shrink to fill the capacity of the container they are in.

Image dimensions of the following image are: 883px x 431px.

This size is used for image carousels on landing pages (e.g. departmental home pages).

Image dimensions of the following image are: 590px x 288px.

This size is used for full bleed inner page leading images (images that appear typically at the top of content pages). Note that this example and the one above it show at the same width as they have been sized to fit inside the width of the div in which they sit.

Image dimensions of the following image are: 250px x 250px.

Inline images set to float left and right

This image style should be used in the body of articles or on long pages of content. Examples for this could be news articles, or long feature articles. The key feature of displaying images using this approach is that text can wrap around them.

Styleguide notes

Image handling

Usage

There are two ways to implement stretching images for responsive layout within a standard article page.

Images can be dropped inline and floated left or right or scaled inside of a grid column to allow for greater control over text and image position for more complex layouts.

You should use the inline images within the body of a long article page, and floating resizable images on listing pages (like events listings).

When creating new images for the site please follow the image dimensions set out below:

Teaser images

W: 250px

H: 250px

Promo images

W: 349px

H: 125px

Department hero carousel images

W: 883px

H: 431px

Small events/ search results/profile pic images

W: 153px

H: 153px

Full bleed inner page leading image

W: 590px

H: 288px

Interaction

N/a

CSS and media queries

A media query is set to stop Inline floating images below 1280 resolution to prevent thin columns of text becoming illegible.

Images can be set to scale with the fluid grid and can also be floated left or right within a column of body copy. Floated images have padding set to allow a gutter between the edge of the image and the text. On mobile resolutions media queries are used to reduce the padding to maximise screen real estate.
